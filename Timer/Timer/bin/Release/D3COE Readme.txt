简易说明
①点击职业更换元素循环。点Reset按钮或者按快捷键可以让元素跳到下一个并重置计时（到底哪种方式更便捷有待进一步研究……），用于对齐时间。
②双击当前元素文字可以缩小/放大窗口。可以拖动窗口，右键点击弹出菜单可以让窗口总在最前。
③默认F11是快捷键，用于刷新时间对齐戒指。点进那个文本框，键盘上按别的键可以更改，有时可能和其他的快捷键冲突，不好使时候多试试找个能用的吧。
④“穿越”按钮（Ctrl+快捷键）可以让窗口变透明且悬浮于窗口/最大化模式的暗黑3。再按Ctrl+快捷键可以恢复到正常模式。

文件夹里有各种wav文件，就是声音文件，不要声音就删除，要声音就留着，也可以换成自己的配音，需要wav格式。从wow-DBM插件提出的夏一可的3、2、1倒数再拼合在一起的，4秒的长度。在备用声音文件夹里。
